## Description

Tiptap 에디터로 글 작성 페이지 구현

### 작업 내용
- 제목 입력 필드
- 카테고리 선택 (드롭다운)
- Tiptap 에디터
- isPublic 토글
- 저장 버튼
- POST /posts 호출

## Acceptance Criteria

- [ ] 제목, 카테고리, 내용, isPublic 입력

- [ ] 저장 성공 → /posts/:slug-:id 이동
- [ ] dirty state 관리 (수정사항 있으면 버튼 활성화)
