## Description

모바일/태블릿/데스크톱 반응형 디자인

### 작업 내용
- 모바일(≤768px), 태블릿(769-1024px), 데스크톱(≥1025px)
- LSB: 모바일에서 햄버거 메뉴
- RSB: 모바일/태블릿에서 숨김
- 글 카드: 모바일 1열, 태블릿 2열, 데스크톱 3열
