# Story 2.4: Passport GitHub OAuth 구현

## 메타 정보
- **Epic**: Epic 2 - 데이터베이스 및 인증 구현
- **Priority**: High
- **Estimate**: 2.5일 (20시간)
- **Dependencies**: 2.3 완료

---

## Description

GitHub OAuth 로그인 및 JWT 토큰 발급 구현

### 작업 내용
- GitHub OAuth App 생성 (clientId, clientSecret)
- Passport GitHub Strategy 구현
- GET /auth/github (리다이렉트)
- GET /auth/github/callback (토큰 발급)
- User 생성/조회 로직

---

## Acceptance Criteria

- [ ] GET /auth/github → GitHub 로그인 페이지 리다이렉트
- [ ] 로그인 성공 → JWT 토큰 발급
- [ ] User 테이블에 githubId, githubUsername 저장
- [ ] 기존 GitHub 사용자 로그인 시 새 User 생성 안 함
