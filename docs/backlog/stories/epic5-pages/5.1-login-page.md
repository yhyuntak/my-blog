## Description

ID/PW 로그인 및 GitHub OAuth 로그인 UI 구현

### 작업 내용
- ID/PW 로그인 폼
- GitHub OAuth 버튼
- POST /auth/login 호출
- JWT 토큰 localStorage 저장
- 로그인 성공 → 홈으로 이동

## Acceptance Criteria

- [ ] Username, Password 입력 → 로그인 성공
- [ ] 잘못된 비밀번호 → 에러 메시지 표시
- [ ] "GitHub으로 로그인" 버튼 → GET /auth/github 리다이렉트
- [ ] 로그인 성공 → useAuthStore 업데이트
- [ ] 이미 로그인 시 `/login` 접근 → 홈으로 리다이렉트
