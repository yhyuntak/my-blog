## Description

3-column 레이아웃 글 상세 페이지

### 작업 내용
- GET /posts/:id 호출
- 3-column 레이아웃 (LSB + 본문 + RSB)
- 본문: Tiptap JSON → HTML 렌더링
- RSB: 목차(TOC) 생성 (h1, h2, h3 추출)
- 관련 글 표시 (같은 카테고리 최근 3개)
