## 주요 기능

- Hero Section: 블로그 소개
- 카테고리별 최근 글 4개씩 표시
- 카드 형태 (썸네일 + 제목 + 요약 + 날짜)
- "더보기" 클릭 → `/posts?category=slug`

## 데이터 로딩

```typescript
// 각 카테고리별 최근 글 4개씩
const techPosts = useQuery(['posts', 'tech', { limit: 4 }])
const lifePosts = useQuery(['posts', 'life', { limit: 4 }])
const projectPosts = useQuery(['posts', 'project', { limit: 4 }])
```
