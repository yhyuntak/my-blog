# 카테고리 관리 페이지 (`/admin/categories`)

## 레이아웃

```
┌────────────────────────────────────────────────────────────────────┐
│ Header (Sticky)                                                    │
└────────────────────────────────────────────────────────────────────┘
┌──────────┬─────────────────────────────────────────────────────────┐
│  Left    │  카테고리 관리                                           │
│ Sidebar  │                                                         │
│          │  ┌───────────────────────────────────────────────────┐ │
│ 카테고리  │  │ Root                                              │ │
│ 목록     │  │                                                   │ │
│          │  │ ▼ 📚 기술 (15개 글)         [수정] [삭제→휴지통]  │ │
│          │  │     ├─ React (5개)        [수정] [삭제→휴지통]  │ │
│          │  │     └─ NestJS (3개)       [수정] [삭제→휴지통]  │ │
│          │  │                                                   │ │
│          │  │ ▼ 🌱 일상 (20개 글)         [수정] [삭제→휴지통]  │ │
│          │  │ ▼ 🚀 프로젝트 (8개 글)      [수정] [삭제→휴지통]  │ │
│          │  │                                                   │ │
│          │  │ [+ 새 카테고리 추가]                              │ │
│          │  └───────────────────────────────────────────────────┘ │
│          │                                                         │
│          │  ┌───────────────────────────────────────────────────┐ │
│          │  │ 🗑️ Trash (휴지통)                                 │ │
│          │  │ ○ 삭제된 카테고리 1 (2개 글)   [복구] [영구삭제]  │ │
│          │  └───────────────────────────────────────────────────┘ │
└──────────┴─────────────────────────────────────────────────────────┘
```

## 주요 기능

- 계층 구조 트리 표시 (접기/펼치기)
- 드래그 앤 드롭으로 순서 변경
- 카테고리 추가/수정/삭제
- 삭제 → Trash로 이동
- Trash에서 복구 or 영구 삭제
- Depth 제한 검증 (최대 3단계)
