# 글 작성/수정 페이지

## 글 작성 (`/posts/new`)

```
┌────────────────────────────────────────────────────────────────────┐
│ Header (Sticky)                                                    │
└────────────────────────────────────────────────────────────────────┘
┌────────────────────────────────────────────────────────────────────┐
│  제목: [                                                      ]     │
│  카테고리: [기술 ▼]    공개: [● 공개  ○ 비공개]                     │
├────────────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────────────┐ │
│  │  Tiptap 에디터 (노션 스타일)                                  │ │
│  │                                                              │ │
│  │  타이핑하면 실시간으로 렌더링됨                                │ │
│  │  **굵게** → 굵게                                             │ │
│  │  # 제목 → 제목 스타일 적용                                    │ │
│  │  ```typescript                                               │ │
│  │  코드 블록 (syntax highlighting)                             │ │
│  │  ```                                                         │ │
│  └──────────────────────────────────────────────────────────────┘ │
├────────────────────────────────────────────────────────────────────┤
│  [요약 생성 (LLM)] [미리보기] [취소] [저장]    💾 자동저장 중...   │
└────────────────────────────────────────────────────────────────────┘
```

## 주요 기능

- 제목, 카테고리, 공개/비공개 선택
- Tiptap 에디터 (전체 화면)
- 자동저장 (debounce)
- LLM 요약 생성 버튼
- 미리보기 모달
- 저장 → `/posts/:slug-:id`로 이동

## 글 수정 (`/posts/:id/edit`)

- 글 작성과 동일한 UI
- 기존 데이터로 채워짐
- Tiptap JSON 불러와서 에디터에 복원
