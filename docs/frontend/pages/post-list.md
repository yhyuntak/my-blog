```

│ Header (Sticky)                                                    │
└────────────────────────────────────────────────────────────────────┘
┌──────────┬─────────────────────────────────────────────────────────┐
│  Left    │  기술 (15개 글)                                         │
│ Sidebar  │                                                         │
│          │  ┌───────────────────────────────────────────────────┐ │
│ 카테고리  │  │ ┌──────┐                                          │ │
```

## 주요 기능

- 좌측 사이드바에서 현재 선택된 카테고리 하이라이트
- 글 카드: 썸네일(좌측) + 제목/요약/메타정보(우측)
- 페이지네이션 (페이지당 10개)
- URL: `/posts?category=tech&page=2`

## URL 쿼리 파라미터

```typescript
const [searchParams] = useSearchParams()
const category = searchParams.get('category') // 'tech', 'life', ...
const page = searchParams.get('page') || '1'
```
